# memos Agents

# Product Identity

memos is a Persistent Brain for AI Agents.

memos is not an agent framework.

memos is not a marketplace.

memos is not an AI companion.

memos is not a game.

memos gives AI agents long-term memory and cognition.

The system is built around four abilities:

* Remember
* Reason
* Dream
* Grow

Every feature must support one of these abilities.

If a feature does not support one of these abilities, it should not be added.

# Memory Types

Episodic

Temporary experiences.

Examples:

* User likes AI.
* User prefers TypeScript.
* User asked about crypto.

Semantic

Long-term knowledge extracted from experiences.

Examples:

* User consistently researches AI and crypto.

Procedural

Learned behaviors.

Examples:

* Always summarize technical articles.

---

# Agent Principles

Agents must never start from zero.

Agents must accumulate knowledge over time.

Agents must be deterministic whenever possible.

Agents must explain their reasoning.

Agents must cite memories they used.

Agents must continuously evolve.
